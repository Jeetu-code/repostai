🚀 RepostAI – Repurpose Content for Social in Seconds

RepostAI is a clean, modern single-page web application that helps users instantly transform long-form content into ready-to-post social media formats like X threads, LinkedIn posts, and Instagram captions.

Built with a scalable SaaS mindset using React + Vite + Tailwind CSS, and deployed on Vercel.

✨ Live Demo

🔗 Deployed on Vercel
(https://repostai-one.vercel.app/)


🧠 Problem

Creators often write long-form content (blogs, notes, scripts, essays) but struggle to:

Repurpose it for multiple platforms

Maintain format differences (X threads vs LinkedIn vs Instagram)

Do it quickly

RepostAI solves this in seconds.

🧑‍💻 Local Development

Clone the repository:

git clone https://github.com/Jeetu-code/repostai.git

Install dependencies:

npm install

Run development server:

npm run dev

Build for production:

npm run build
🚀 Deployment (Vercel)

Push to GitHub

Import project in Vercel

Select Vite preset

Deploy

No environment variables required (for current MVP).
